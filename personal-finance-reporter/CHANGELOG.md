# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-01

### Added
- Initial release
- CSV parser with validation (`src/parser.py`)
- Financial analytics engine (`src/analytics.py`)
- Chart generation with matplotlib (`src/charts.py`)
- PDF report generation with ReportLab (`src/report.py`)
- CLI entry point with argparse (`main.py`)
- Sample dataset (`data/expenses_sample.csv`)
- Full test suite with pytest (`tests/`)
- Bilingual README (English / Deutsch)
